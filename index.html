  • Place each ship in any horizontal or vertical position but not diagonally.
  • Position of any ships cannot be changed once the game has begun.
  • Ship sizes are
    • Carrier: 5 tiles
    • Battleship: 4 tiles
    • Destroyer: 3 tiles
    • Submarine: 3 tiles
    • Patrol Boat: 2 tiles
  • The game is turn based. Pick a tile on the opponent's board to attack.
  • Game ends when one side loses all 5 ships
